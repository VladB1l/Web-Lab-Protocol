Створити файл index.html і дві папки: CSS та JS. У папці CSS створити файл style.css. У папці JS створити файл gallery.js. У свій JavaScript файл додай масив зображень. Кожний об’єкт являє собою один елемент галереї. Виконати делегування, а саме додати функціонал прослуховування кліка по елементах галереї та отримання посилання на велике зображення при кліку. Для цього використовуй прийом делегування на ul.gallery. Поки що при кліку на елемент галереї виводь у консоль посилання на велике зображення. 11. Підключення бібліотеки. Бібліотека basicLightbox представляє повністю функціональне модальне вікно, яке відмінно підходить під поставлену задачу. Використовуйте CDN сервіс jsdelivr і додай в HTML файл посилання на мініфіковані (.min) JS та CSS файли бібліотеки.

            const images = [];

                newObject.preview = `./img/picture${i + 1}.jpg`;

                newObject.description = `picture${i + 1}`;
                images.push(newObject);

            let gallery = document.querySelector('.gallery');

                let newElement = document.createElement('li');
                newElement.innerHTML = `img src=${'images[i].preview}' alt='${images[i].description}';
                newElement.onclick = () => {
                    basicLightbox.create(`
                        img width="1400" height="900" src="${images[i].original}"
                    `).show();
                };
                gallery.append(newElement);
