Типи селекторів

  1. Селектор елемента (тега);
  2. Селектор класу;
  3. Селектор ідентифікатор;
  4. Дочірні;
  5. Універсальні селектори;

Селектор тегу

Якщо як селектор виступає ім'я елемента (тега), то цьому випадку всі елементи даного тега будуть відповідати заданим стилям. Якщо всередині цих тегів будуть текстові елементи, вони успадкуватимуть певні правила. 


            h1 {
                text-transform: uppercase;
            }
            <h1>приклад заголовку </h1>

приклад заголовку

Селектор класу

Якщо потрібно встановити стилі для індивідуального елемента або задати різні стилі для одного тега слід використовувати селектор класу. Імена класів повинні починатися з латинського символу і можуть містити символ дефісу і підкреслення. Щоб прив'язати до елемента клас, необхідно у тезі, що відкриває, задати атрибут class і привласнити йому ім'я класу, описаного в стилях 

 <ul class="nav">
    <li><a href="">Free-To-Play</a></li>
    <li><a href="">Latest Games</a></li>
    <li><a href="">Coming Soon</a></li>
    </ul>

    <style>
    .nav {
    list-style: none;
    }
    </style>
  • Free-To-Play
  • Latest Games
  • Coming Soon

Селектор ідентифікатора

Ідентифікатор – особливий вид селектора, яким позначається один елемент на сторінці. Щоб вказати селектор ідентифікатор, у CSS необхідно використовувати конструкцію #ім'я_ідентифікатора. 


        <p id="example"> Lorem, ipsum dolor sit.</p>
       
            #example{
                color:darkgoldenrod;
                font-size: 20px;
            }

Lorem ipsum dolor sit.

Дочірні та контекстні селектори

Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елементу. 


        .productListItem button {
        font-size: 20px;
        background-color: #6a8704;
        padding: 8px 55px;
        color: white;
    }
    <div class="productListItem">
        <button><a href="">Play Now</a></button>
    </div>

Універсальні селектори - це селектори CSS, які відповідають будь-якому елементу на веб-сторінці. Вони використовуються для застосування стилів до всіх елементів на сторінці без вказівки конкретного типу або класу елементу. Ось кілька прикладів універсальних селекторів:


        * {
           CSS стилі
        }
